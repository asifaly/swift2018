MT 341 Format Specifications

The MT 341 consists of four sequences:

  • Sequence A General Information contains general information about the FRA as well as about the confirmation itself

  • Sequence B Transaction Details contains information about the transaction

  • Sequence C Settlement Instructions for the Settlement Amount provides the settlement instructions.

  • Sequence D Reporting Information provides transaction information that must be reported to a trade repository. This sequence is optional and is only to be used for reporting to trade repositories, although some fields, for example, unique transaction identifier and prior unique transaction identifier, might be used on regular confirmations.
