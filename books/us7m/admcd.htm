MT 759 Network Validated Rules

C1 

If field 23H contains ISSUANCE, REQISSUE, REQAMEND, or ISSAMEND, then field 22D must contain UNDK and if field 23H contains TRANSFER, then field 22D must contain DGAR, STBY, or UNDK (Error code(s): D87).

If field 23H is ... Then field 22D must contain ...
ISSUANCE, REQISSUE, REQAMEND, or ISSAMEND UNDK
TRANSFER DGAR, STBY, or UNDK
CLSVOPEN, CLSVCLOS, FRAUDMSG, GENINFAD, OTHERFNC, REIMBURS, or REQFINAN DGAR, DOCR, STBY, or UNDK
